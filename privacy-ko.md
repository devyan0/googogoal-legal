# GooGoGoal 개인정보처리방침

**시행일: 2026년 6월 7일**

GooGoGoal(이하 "앱")은 목표와 습관을 만들고 기록하도록 돕는 iOS 앱입니다. 이 문서는 앱이 어떤 정보를 처리하는지, 어떤 목적으로 사용하는지, 사용자가 어떤 선택권을 갖는지 설명합니다.

앱은 현재 회원가입 없이 사용할 수 있으며, 사용자가 입력한 목표와 기록은 기본적으로 사용자의 기기 안에 저장됩니다.

---

## 1. 개인정보의 처리 목적

앱은 다음 목적으로 개인정보를 처리합니다. 처리하는 개인정보는 아래 목적 이외의 용도로는 이용되지 않으며, 목적이 변경되는 경우 사전에 동의를 받는 등 필요한 조치를 합니다.

| 처리 목적 | 내용 |
|---|---|
| 서비스 제공 및 운영 | 목표와 습관 기록 기능 제공, 앱 안의 진행 상황 표시 |
| AI 목표 추천 | 사용자가 입력한 목표 관련 자유 입력 텍스트를 바탕으로 측정 가능한 지표를 추천 |
| 구독 결제와 권한 관리 | 유료 구독 결제, 구매 복원, 프리미엄 권한 확인 |
| 서비스 개선과 통계 분석 | 기능 사용 패턴 분석, 오류 진단, 사용성 개선 |

---

## 2. 처리하는 개인정보의 항목

앱은 회원가입을 받지 않으며 이름, 전화번호, 주소 같은 직접 식별정보를 수집하지 않습니다. 처리하는 항목은 다음과 같습니다.

| 구분 | 항목 | 수집 방법 |
|---|---|---|
| 앱 이용 식별자 | 분석과 결제 권한 확인에 필요한 기기 또는 앱 설치 단위의 식별자 | 앱 이용 중 자동 생성 |
| 이용자 입력 데이터 | 목표 제목과 설명, 습관 지표, 체크인 기록, AI 추천을 위한 자유 입력 텍스트(예: 운동이나 생활 목표 설명) | 이용자 직접 입력 |
| 결제 관련 정보 | 구독 상품 식별자, 구매와 갱신 상태, 무료 체험 여부(결제 수단 정보는 Apple이 처리하며 앱은 수집하지 않음) | 결제 진행 시 자동 수집 |
| 서비스 이용 기록 | 기능 사용 이벤트, 화면 이동, 구매 화면 상호작용 등 사용 데이터 | 앱 이용 중 자동 수집 |

앱은 광고 식별자(IDFA)를 사용하지 않으며, 추적(App Tracking Transparency 대상)을 위한 데이터를 수집하지 않습니다.

목표와 습관 기록 등 이용자 입력 데이터는 원칙적으로 이용자의 기기 내부(앱 전용 저장 공간)에만 보관됩니다. 다만 AI 추천 기능을 사용할 때 입력한 자유 입력 텍스트는 아래 4항과 같이 외부로 전송됩니다.

## 3. 개인정보의 보유 및 이용 기간

- 기기에 저장되는 목표와 기록 데이터: 이용자가 앱을 삭제하거나 앱 내에서 데이터를 삭제할 때까지 보관됩니다. 앱을 삭제하면 기기 내 데이터는 함께 삭제됩니다.
- 분석 데이터: 서비스 제공과 통계 분석 목적 달성에 필요한 기간 동안 보관 후 파기합니다.
- 결제와 구독 기록: 관련 법령(전자상거래 등에서의 소비자보호에 관한 법률 등)에서 정한 기간 또는 Apple 및 결제 처리자의 정책에 따라 보관됩니다.
- AI 추천을 위해 외부로 전송된 자유 입력 텍스트: 추천 결과를 생성하는 데 필요한 일시적 처리 목적에 한정되며, 처리 후 보관 기간은 아래 4항의 각 처리자 정책을 따릅니다.

## 4. 개인정보의 제3자 제공 및 처리 위탁

앱은 원활한 서비스 제공을 위해 다음과 같이 개인정보 처리 업무를 외부 사업자에게 위탁합니다. 위탁받는 사업자는 위탁 목적 범위에서만 개인정보를 처리합니다.

| 처리자(수탁자) | 위탁 업무 | 처리 항목 |
|---|---|---|
| Supabase, Inc. | AI 추천 요청 처리와 서버 기능 운영 | AI 추천 요청 정보, 기술 요청 정보 |
| Amplitude, Inc. | 서비스 이용 통계와 분석 | 기기 식별자, 서비스 이용 기록 |
| RevenueCat, Inc. | 구독 결제와 권한 처리 | 구독 상품 식별자, 구매와 갱신 상태 |
| OpenRouter, Inc. (경유) / Google LLC / OpenAI, L.L.C. | AI 목표 추천 생성(자유 입력 텍스트를 대형 언어 모델로 전송하여 추천 결과 생성) | AI 추천을 위한 자유 입력 텍스트, 카테고리, 언어 설정 |

- AI 추천 기능을 사용하면 이용자가 입력한 자유 입력 텍스트가 OpenRouter를 경유해 Google Gemini 또는 OpenAI 모델로 전송될 수 있습니다. 이때 이름, 이메일, 전화번호 같은 직접 식별정보는 함께 전송하지 않습니다.
- 위탁 사업자가 변경되거나 위탁 업무 내용이 바뀌는 경우 이 처리방침을 통해 공개합니다.

## 11. 자동화된 AI 처리에 관한 안내

앱의 AI 목표 추천 기능은 이용자가 입력한 자유 입력 텍스트를 외부 대형 언어 모델(Google Gemini, OpenAI 모델 등)로 전송하여 측정 가능한 목표 지표를 자동으로 생성해 제안합니다. 이 추천은 참고용이며, 어떤 목표와 지표를 사용할지는 이용자가 직접 선택합니다. 추천 결과 생성 과정에서 직접 식별정보는 전송하지 않습니다.
